
# Sistema de Gestión Web

Sistema web desarrollado como proyecto académico y personal, orientado a la gestión de productos y procesos, utilizando tecnologías backend y frontend.

## 🛠️ Tecnologías utilizadas
- PHP.
- MySQL.
- JavaScript.
- jQuery.
- HTML5.
- CSS3.

## 📋 Funcionalidades principales.
- Gestión de productos (alta, baja, modificación).
- Gestión de usuarios (alta, baja, modificación).
- Conexión a base de datos MySQL.
- Formularios dinámicos.
- Validaciones en frontend.
- Estructura modular del sistema.
- Interfaz web responsive básica.

Requerimientos:
Se nesecita tener instalado xampp y la carpeta "cotillon" guardada htdocs.
El archivo "cotillon.sql" cargado en la base de datos MySQL en una tabla llamada "cotillon".
